# Active-Directory-
Active Directory lab

## Overview
This project demonstrated the deployment and configuration of a Windows Active Directory Environment

## Technologies Used
- Windows Server 2022
- Windows 11
- Active Directory Domain Services
- Group Policy
- DNS / DHCP
- VMware
- PowerShell

## Lab Objectives 
- Deploy a Domain Controller
- Join Client machines to the domain
- Create and manage users and groups
- Appply Group Policy Objects
- Enforce Security Best practices
- Troubleshoot common AD Issues

## Key Skills Demonstrated
- Active Directory Adminstration
- Identity and Access Management
- Group Policy Configuration
- Windows Troubleshooting
- Security Hardening
